OpenCV stands for "Open Source Computer Vision"

[How to install Opencv for Python]
If you haven't installed OpenCV yet, please do so by following the instructions below. OpenCV was added lately in the online Python package repository so first make sure you have the latest version of pip by executing:
# pip install --upgrade pip

Then install OpenCV:

# pip install opencv-python

And then you can import opencv in Python as:

import cv2
